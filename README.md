## 阿柠檬桌面端

https://alemonjs.com/

跨平台开发的事件驱动机器人客户端

### 本地打包

```sh
git clone --depth==1 https://github.com/lemonade-lab/alemonjs-desktop.git
```

```sh
# 使用 nvm切换版本
nvm use 22
# 安装 yarn
npm install yarn@1.19.1 -g
# 安装所有依赖
yarn install
```

```sh
# 编程开发
yarn dev # mac
yarn dev-win # win
```

```sh
# 生产图标
yarn icons
# 打包
yarn build
```
